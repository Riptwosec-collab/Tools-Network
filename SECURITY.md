# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it via [GitHub Issues](https://github.com/Riptwosec-collab/Tools-Network/issues) with the label `security`.

We will respond within 48 hours.

## Scope

- XSS vulnerabilities in the web interface
- CSP bypass issues
- Privacy leaks

## Out of Scope

- Issues with third-party tools linked from this app
- Social engineering attacks
